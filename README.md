# Strassen Algorithm – Java

Implementation and comparison of two matrix multiplication algorithms:
- **Classic** matrix multiplication (O(n³))
- **Strassen's** recursive algorithm (O(n^2.807))

## How it works
Two random 4x4 matrices A and B are generated.
Both algorithms compute the product and the total
number of operations performed by Strassen is printed.

## How to run
javac src/*.java
java -cp src Main

## Language
Java
